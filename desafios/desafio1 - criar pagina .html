O que é e para quê serve o HTML?

Voltar

HTML é a linguagem base para criar páginas web e define a estrutura e o conteúdo do conteúdo na internet. Serve para estruturar diversos se não todos os elementos apresentados num Navegador Web.

Comunicação Clients x Servers

Voltar
  1. Clientcliente Em computação, um cliente é um dispositivo ou software que solicita serviços ou recursos para um outro dispositivo, neste caso, o servidor.
  2. Serverservidor Em computação, um servidor é o dispositivo ou sistema fornecedor dos serviços, recursos ou dados solicitados pelo cliente. neste caso, o servidor.

Em resumo, o modelo de comunicação client-server é uma arquitetura de rede onde os clientes se conectam a um servidor, para acessar recursos ou serviços.

É um modelo amplamente utilizado em diversas aplicações, como na internet, redes locais e em sistemas de computação distribuída.

Semântica HTML

Voltar
linguísticaSemântica, se remete ao sentido das palavras e da interpretação das sentenças e dos enunciados.

No contexto do HTML, a semântica se refere ao uso de tags que descrevem o tipo e o significado do conteúdo que contém, em vez de apenas delimitar visualmente áreas da página.

Algumas Tags

Voltar

  • "del" - utilizada para se referir ao conteúdo removido de um documento, por exemplo esta parte foi removida do texto
  • "video/img/audio" - tags de mídia, utilizadas para inserção de recursos multimídia na página.
  • "h6" - a ultima opção para header ou "cabeçalho", sendo a:
    menor existente
